
# 🌱 Transformer-Based Weed Detection (SegFormer)

## Overview
This project uses a Transformer-based model (SegFormer) for semantic segmentation to classify crops and weeds.

## Tech Stack
- Python
- PyTorch
- Hugging Face Transformers
- OpenCV
